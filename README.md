# Princess VS Monsters

**Princess VS Monsters** is a 2D platformer game developed in Python using the Pygame library.  
The player controls a wizard princess, collects coins and keys, navigates platforms and stairs, and defeats enemies using a magic projectile.

## Status

Completed

## Features

- 2D platformer level based on a tile map
- Player movement system (left/right + stair navigation)
- Coin collection system
- Key interaction and object interaction mechanics
- Enemy movement and collision handling
- Shooting mechanic using magic projectiles
- Camera system following the player
- Sound effects for actions (shooting, collecting, interaction)
- Basic collision system with platforms and obstacles

## Technologies Used

- Python  
- Pygame  

## Controls

| Key | Action |
|---|---|
| A | Move left |
| D | Move right |
| W | Move up (stairs) |
| S | Move down (stairs) |
| E | Interact (keys, objects) |
| Space | Shoot |
